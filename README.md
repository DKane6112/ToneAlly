# ToneAlly
A tool for assisting musicians with music theory and song writing created with Java (SpringBoot), HTML, CSS and JavaScript.

## Create
Get inspired and create new music using ToneAlly's create feature.
![ToneAllyPreview1](https://github.com/DKane6112/ToneAlly/assets/93456651/21a85abe-0402-444f-8768-7a38a4f3f6f8)

This feature takes information from the user about what kind of music they would like to create (the key, mood and genre) and returns a suitable chord progression along with the notes in different scales of the user's selected key.
![ToneAllyPreview2](https://github.com/DKane6112/ToneAlly/assets/93456651/44567268-2469-472b-bfdf-c1bf206decb2)

Chords and scales can then be clicked to display an image of how to play it using chord diagrams and tablature.
![ToneAllyPreview3](https://github.com/DKane6112/ToneAlly/assets/93456651/a47cee93-9414-4809-bb07-e2c02960fa47)
